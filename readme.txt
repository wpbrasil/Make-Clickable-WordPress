== Description ==

Convert plain text URI to HTML links. Converts URI, www, FTP, and email addresses.

= Usage =

1. Download and extract `make-clickable-wp.zip` to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Installation ==

1. Download and extract `wp-smtp.zip` to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
